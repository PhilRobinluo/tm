# tm - tmux 交互式引导包

> 不背命令也能用 tmux。用着用着，就把命令学会了。

## 这是什么

`tm` 是一个 tmux 的交互式引导工具。它把 tmux 所有常用操作包装成中文菜单，让你按数字就能操作。

**核心设计：每次操作后，自动告诉你对应的原始命令。**

用了 10 次，你不知不觉记住了命令。
用了 30 次，你直接输命令，引导包退休。

## 安装（一条命令）

```bash
bash install.sh
```

自动完成 4 件事：
- 检查 tmux，没装帮你装（支持 Mac/Linux）
- 安装 `tm` 命令到 PATH
- 配置 tmux（鼠标支持、历史记录、快捷键速查）
- 添加终端启动提示

已有配置会自动备份，重复运行不会搞乱。

## 使用

```bash
tm          # 打开交互式管理菜单
tm new      # 创建新的工作空间
tm a        # 快速进入已有空间
tm ls       # 查看所有工作空间
tm d        # 离开当前空间（后台保持运行）
tm keys     # 快捷键速查
tm help     # 查看所有命令
```

### 三个阶段

| 阶段 | 你会怎么用 |
|------|-----------|
| 刚开始 | `tm` 打开菜单，按数字操作 |
| 用了一周 | `tm a`、`tm new`、`tm ls` 快捷命令 |
| 用了一个月 | 直接 `tmux attach`、`tmux new -s work`，引导包退休 |

## 终端启动提示

安装后每次打开终端会看到：

```
  🖥️  tmux: 2 个工作空间在运行  (main, code)
  ──────────────────────────────────────────
  tm a  进入空间     tm new  创建新空间
  tm ls 查看全部     tm      打开管理菜单
```

不阻塞、不强制。只是告诉你当前状态。

## tmux 快捷键速查

在 tmux 里按 `Ctrl+B → h` 弹出速查浮窗（安装时自动配置）。

或者在终端输入 `tm keys` 查看分区速查卡。
