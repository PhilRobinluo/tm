# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [1.0.0] - 2026-03-20

### Added
- tmux 交互式会话管理（创建/进入/列出/离开/关闭）
- 快捷键学习模式（联想记忆法）
- 工具箱：端口管理、进程管理、快速搜索、网络诊断、系统状态
- Git 操作中心（9 大分类菜单，30+ 快捷命令）
- 项目导航（基于 zoxide 快速跳转）
- 文件权限管理、压缩/解压
- teach() 教学系统 — 每次操作自动展示对应的原始命令
- 配置文件支持（~/.tmrc）
- 插件系统（~/.tm/plugins/）
- `tm update` 一键升级
- `tm --version` 版本查询
- Homebrew 安装支持
- GitHub Actions CI（ShellCheck 语法检查）
